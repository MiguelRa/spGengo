spGengo SharePoint Plugin (full package)

This package contains:

- spGengo_bin package: binary components for  manually deploying spGengo plugin.  

- spGengo_setup package: installers for automatically deploying spGengo plugin.

- spGengo_src package: spGengo source code.

- spGengo_demo.avi: demo video demonstrating the usage of spGengo plugin.

- spGengo_Quick_Guide.pdf: quick installation and usage guide.
